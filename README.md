# Unity-chan alerm

## 概要

* Unity-chan を使ってアラームを作りました

## レポジトリのclone方法

Windowsで C:\git フォルダで運用を想定している。他のフォルダで使っている場合は、読み替えること。

    cd /d c:\git
    git clone https://github.com/lastshogun13/Unity-chan-alerm.git

## ブランチ運用

1. master
基本的にはここを使う

2. 他のブランチ
今は運用予定なし
